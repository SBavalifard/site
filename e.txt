 git init     ساخت پوشه ی .git

git help       توضیحات گیت

git status   دیدن وضعیت فایل ها

git add e.txt     بعد از ساخت فایل مثلاe.txt آن را از ویندوز به روی صحنه میبریم

git commit -m ""poroject initialized and e.txt created"    بعد از کامیت باید با دش ام یک پیغام بنویسیم که اجباری هست و ما با آن می فهمیم در این کامیت چه اتفاقی افتاده   

در صورتی که در محتوای فایل تغیراتی ایجاد کردیم آن فایل از روی صحنه می رود و با دستور بعدی(خط پایین) آن را روباره برمی گردانیم

git add.  با این دستور دوباره روی صحنه میرود و تا کامیت نکنیم نمی توانیم پوش کنیم

با کامیت مجدد باید با داش ام یک پیغام هم بگداریم
